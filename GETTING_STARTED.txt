METHOD 1: DOCKER (FASTEST - RECOMMENDED)
-----------------------------------------
1. Open PowerShell in the project directory
2. Run: docker-compose up -d
3. Wait for services to start (2-3 minutes)
4. Open browser: http://localhost
5. Login with demo@budgetmate.com / Demo@123

METHOD 2: LOCAL DEVELOPMENT
----------------------------
1. Open PowerShell in the project directory
2. Run: .\setup.ps1
3. Start database: docker-compose up -d db redis
4. Setup database:
   cd api
   npm run prisma:migrate
   npm run prisma:seed
5. Start backend (Terminal 1):
   cd api
   npm run dev
6. Start frontend (Terminal 2):
   cd web
   npm run dev
7. Open browser: http://localhost:5173
8. Login with demo@budgetmate.com / Demo@123

================================================================================
TROUBLESHOOTING
================================================================================

Port Already in Use:
- Check: netstat -ano | findstr :5000
- Solution: Change port in docker-compose.yml or .env

Database Connection Failed:
- Check: docker-compose logs db
- Solution: docker-compose restart db

API Not Responding:
- Check: docker-compose logs api
- Solution: Ensure migrations ran, restart API

Frontend Build Errors:
- Solution: Delete node_modules, run npm install
